# 기획의도

코로나 시기에 떠나지 못하는 일상에서 벗어나 여행을 떠난 기분을 느낄 수 있는 여행 플랜 사이트를 만들고자 하였고,
그 중, 해외여행을 가지 못하는 지금 사람들이 많이 찾는 관광지인 제주도를 지역으로 선정하였다.

# 팀원구성

* 김수민
* 안웅선
* 이영훈
* 이유빈
* 조재연
* 최정원

# 개발환경

* Front-end : javascript / jquery / bootstrap
* Back-end : java / spring boot / MyBatis
* DB : Oracle
* OS : windows 10

# 기술구현 내용

* 로그인 / 회원가입
* 제주 액티비티 목록 & 리뷰
* 제주 숙소 목록 & 리뷰
* 제주 여행 플랜 목록 & 리뷰
* 제주 맛집 목록 & 리뷰
* 마이페이지
